MatrixHunter 개인정보처리방침

최종 갱신일: 2026-05-08 · 적용 앱: MatrixHunter (com.matrixhunter.app)

MatrixHunter(이하 “본 앱”)는 사용자의 개인정보를 중요하게 여깁니다. 본 문서는 본 앱이 어떤 정보를 다루는지, 누구와 공유하는지, 그리고 사용자가 가진 권리를 설명합니다. 본 앱은 개인 식별이 가능한 계정, 광고, 추적, 위치 정보, 카메라, 마이크, 연락처를 수집하지 않습니다.

1. 수집·저장하는 정보

본 앱은 사용자의 단말기(디바이스) 내부 저장소(AsyncStorage)에 다음 데이터만 저장합니다. 모든 데이터는 단말기를 떠나지 않으며, 외부 서버로 전송되지 않습니다.

  • @mh:settings — 언어, 사운드, 햅틱 등 사용자 설정값
  • @mh:in_progress_session — 진행 중이던 게임 세션 스냅샷 (앱 재시작 시 이어하기 용도)
  • @matrixhunter:history — 종료된 게임 전적 (최대 50건)
  • @mh:onboarded — 첫 실행 가이드 표시 여부
  • @mh:last_match_id — 직전 종료된 게임 식별자 (홈 화면 바로가기 용도)

이 데이터는 모두 단말기 로컬에만 존재하며, 사용자가 앱을 삭제하거나 앱의 데이터 초기화를 수행하면 함께 제거됩니다.

2. 권한 사용

  • 진동(Vibration) — 게임 효과용 햅틱 피드백
  • 오디오 재생 — 게임 효과음 재생
  • 공유 시트(Share Sheet) — 사용자가 명시적으로 결과 이미지를 공유할 때만 호출됨
  • 인터넷 — 아래 “3. 제3자 서비스” 항목에 명시된 경우 한정

본 앱은 위치, 카메라, 마이크, 연락처, 광고 식별자(AAID), 외부 저장소 파일 접근 권한을 요구하지 않습니다.

3. 제3자 서비스

3.1 Sentry (오류 진단) — 선택적 활성화

본 앱은 프로덕션 빌드에서 충돌·오류 진단을 위해 Sentry SDK (@sentry/react-native)를 통합할 수 있습니다. 이 기능은 빌드 시점의 환경 변수 EXPO_PUBLIC_SENTRY_DSN이 설정된 경우에만 활성화되며, 활성화 시 다음과 같은 정보가 Sentry로 전송될 수 있습니다.

  • 오류 스택 트레이스 및 발생 화면 이름
  • 운영체제 버전, 앱 버전, 단말기 모델
  • 충돌 직전의 비식별 이벤트 로그

본 앱은 사용자의 이름, 이메일, 위치, 광고 식별자를 Sentry로 전송하지 않습니다.

3.2 Expo Updates (OTA 업데이트) — 선택적 활성화

본 앱은 expo-updates 모듈을 통한 OTA 업데이트를 지원할 수 있습니다. 이 기능이 활성화되어 있는 빌드에서는 앱 실행 시 EAS Update 서버에 업데이트 매니페스트 조회 요청이 발생할 수 있으며, 이 과정에서 Expo가 정한 비식별 메타데이터(런타임 버전, 채널, 플랫폼)가 송신됩니다.

4. 광고 및 추적

본 앱은 광고를 노출하지 않으며, 어떠한 추적기나 분석 도구를 통해 사용자 행동을 외부에 전송하지 않습니다.

5. 아동의 개인정보

본 앱은 13세 미만 아동을 직접적인 대상으로 하지 않습니다. 본 앱은 외부로 어떠한 개인 식별 정보도 수집·전송하지 않으므로 아동의 개인정보를 별도로 처리하지 않습니다.

6. 데이터 보관 및 삭제

본 앱이 저장하는 모든 정보는 단말기 로컬에만 존재합니다. 사용자는 언제든지 다음 방법으로 데이터를 삭제할 수 있습니다.

  • 운영체제의 앱 설정에서 “저장 공간 → 데이터 지우기” 실행
  • 본 앱 삭제 (Uninstall)

7. 개인정보처리방침 변경

본 정책은 법령, 서비스 변경, 기능 추가 시 갱신될 수 있습니다. 변경 시 본 페이지의 “최종 갱신일”이 함께 갱신됩니다.
